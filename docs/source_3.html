Complete text		Section 1(c) - Percentages

A percentage is a kind of value. 100% specifies a percentage with parts parts per hundred.[1]

Note

[1]. This looks a bit odd, but does make sense. Unusually, this notation involves only one numeric piece - the number before the percentage sign. We give this a name, "parts per hundred", in order to be able to convert percentages to ordinary numbers and back.
